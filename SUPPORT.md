# Support

TFWS 3.0 is an engineering alpha. Public support is provided on a best-effort
basis and does not imply production certification, warranty, or a response-time
commitment.

## Where to ask

- Reproducible defects: open a bug report using the GitHub issue form.
- Design or feature proposals: use the feature request form.
- Contribution questions: open an issue and reference `CONTRIBUTING.md`.
- Security vulnerabilities: follow `SECURITY.md` and report privately.

Before opening an issue, search existing issues and review:

- `README.md`,
- `docs/QUICKSTART.md`,
- `docs/IMPLEMENTATION-STATUS.md`,
- `RELEASE-GATES.md`.

## Scope

The repository can provide guidance about the published engineering-alpha code,
specifications, schemas, test vectors, and validation workflow.

The repository does not currently promise:

- production deployment support,
- certification or compliance approval,
- legal, privacy, or cryptographic assurance,
- support for capabilities listed as scaffolded or incomplete,
- recovery of lost keys or data,
- confidential support through public issues.

Reports without enough information to reproduce or evaluate the request may be
closed until the missing information is supplied.
